```

2. Copy AssessmentMatrix.gs
3. Go to Google Apps Script
4. Create new file, paste code
5. Deploy as web app
6. Save URL
```

### Phase 2: Update App (10 min)
```
1. Open APP_INTEGRATION_CHECKLIST.md
2. Copy AssessmentMatrix.js to components/
```
